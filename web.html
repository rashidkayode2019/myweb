Three Markup Languages The Markup languages are of particular interest.
  1. SGML:Standard Generalize Text Language.
    • Is the parent language from which the other two are derived.
    • It is meta language used to define other markup languages.
    • SGML does not allow any hyper text links.
    • SGML requires that you create your own document language rules.
    • The set of rules is called the Document type Definition[DTD].
    • The DTD is generally specified in a separate file which you reference or declare at the beginning of each document.
  2. HTM:Hypertext Markup Language
  3. XML:Xccessible Markup Language
